How to Use the IoT Smart Drop Box?

An IoT-based Smart Drop Box is a secure, tech-driven parcel delivery solution that ensures safe, contactless deliveries using OTP authentication.

User Guide

  • Create an account or log in if you already have one.
  • Click on the "Sign Up" button and set a password.
  • Once signed in, you can track and access deliveries securely.
  • Enter the information asked to unlock the drop box.
  • After retrieving the parcel, lock the drop box.

Delivery Guide

  • Enter your details (Name, Agency, Order ID, and Mobile No.).
  • Receive a **random 6-digit OTP** to unlock the IoT drop box.
  • Enter the OTP to open the drop box.
  • Place the parcel inside the drop box securely.
  • End the process by locking the lid of the drop box.
